# 已知限制

第一阶段支持显式类型的 Nat、Int、Rat、Real，基础算术、比较、∀、∃、→、∧、¬ 和有限
受控自然句。它不支持任意自然语言证明、隐含变量、代词消歧、自动创造证明思路、完整集合
类型、复杂 coercion、依赖类型、ε–δ 极限、测度论、范畴论、泛函分析或复杂微积分。

自然句只支持明确类型的一至两个全称变量及完整“如果…那么…”结构。存在量词目前只自动
构造可从等式直接提取的简单见证。函数应用和集合关系虽有表达式接口，但类型覆盖有限。

Nat 减法与 Nat/Int 除法主动返回歧义；混合数域不自动插入 coercion。证明策略仅覆盖可由
rfl、exact、simp、ring、linarith、positivity、omega、aesop 等稳定 tactic 解决的形状。
Mathlib 更新、复杂 elaboration 错误和最小 import 优化仍需人工处理。
