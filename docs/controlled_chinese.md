# 受控中文规范

## 推荐模板

    # 定理名称
    正数相加仍为正
    # 变量
    x：实数
    y：实数
    # 假设
    hx：x > 0
    hy：y > 0
    # 结论
    x + y > 0

变量必须显式声明为自然数、整数、有理数或实数。假设名称可省略；名称冲突按稳定编号处理。
结构化字段优先；多个结论或冲突字段返回 STRUCTURED_FIELD_CONFLICT，不会选择其中一个。

## 量词与逻辑

支持“存在实数 y，y = x”“对任意自然数 n，n = n”“如果 P，那么 Q”“P 并且 Q”
和“¬ P”。有限自然句支持“对于任意实数 x 和 y，如果 P，那么 Q”。量词必须紧跟类型、
变量、逗号和完整主体；“任意 n”或裸“有”属于歧义/错误。

## 优先级

从结合最紧到最松：幂、一元负号、乘除、加减、比较、合取、蕴含。因而 -x^2 解析为
-(x^2)，而 (-x)^2 保留括号。建议在复杂表达式中显式写括号。

错误示例包括未声明 y、变量 x 无类型、同一 x 同时声明 Nat/Int、Nat 减法意图不明、
Nat 与 Real 直接比较、多个结论、未知“约等于”。自由证明散文、代词指代和不完整量词
不受支持。
